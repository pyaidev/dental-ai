# Что нужно от вас для завершения настройки

- ОГРН организации
- Юридический адрес
- ФИО ответственного за обработку персональных данных

## 2. Тинькофф — подключение оплаты

Для приёма платежей за подписки нужен **Тинькофф Бизнес** аккаунт.

### Шаги:
1. Зайдите в **Тинькофф Бизнес** → Интернет-эквайринг
   https://www.tinkoff.ru/business/acquiring/
2. Подключите интернет-эквайринг (если ещё не подключён)
3. В личном кабинете: **Интеграция** → **API**
4. Скопируйте:
   - **Terminal Key** (идентификатор терминала)
   - **Secret Key** (секретный ключ)

### Пришлите мне:
- Terminal Key (например: `1234567890`)

## 3. Telegram бот — напоминания пациентам

### Как создать бота:
1. Откройте Telegram → найдите **@BotFather**
2. Напишите `/newbot`
3. Введите имя бота: `Odonta Index AI`
4. Введите username бота: `OdontaIndexBot` (или любой свободный)
5. BotFather выдаст **токен** — длинная строка вида: `7123456789:AAH...`

### Пришлите мне:
- **Bot Token** (например: `7123456789:AAHxyz...`)
- **Username бота** (например: `@OdontaIndexBot`)

### Как работает:
- Врач вносит телефон пациента в карту
- Пациент получает ссылку на бота (в PDF-отчёте или лично)
- Пациент нажимает "Старт" → делится номером телефона
- Бот автоматически находит пациента по номеру
- Автоматические напоминания: контрольная (2 недели) и плановая гигиена

## 4. Max (бывший VK Мессенджер) — напоминания пациентам

Max (https://max.ru) — мессенджер от VK для общения и решения задач.

### Как создать бота для Max:
1. Зайдите на портал разработчика: https://dev.max.ru
2. Зарегистрируйте организацию
3. Создайте нового бота
4. Пройдите модерацию
5. Получите **токен бота**

### Пришлите мне:
- **Max Bot Token**

### Как работает (аналогично Telegram):
- Пациент находит бота в Max и пишет `/start`
- Отправляет свой номер телефона
- Бот автоматически привязывает пациента
- Напоминания приходят в Max

**API Max:** `https://platform-api.max.ru`
**Документация:** https://dev.max.ru/docs-api
